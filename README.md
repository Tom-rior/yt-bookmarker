## About
The YTBookmarker extension helps you save time when watching a YouTube Video. It makes every video on YouTube behave like a book where you can easily bookmark a specific spot. After bookmarking a spot you can easily navigate to this individual timestamp whenever you want. <br><br>
All bookmarks are synced with your Google Chrome account, so you can view the bookmarks when you come back to a previously watched video. <br>

The extension currently works only on Google Chrome.

## Installation and Use Guide
### Installation
Currently the extension is not available on the Chrome Web Store. <br>
To download and install it you have to follow these steps:
1. Make sure <b>you have Google Chrome installed</b> on your computer. The extension <b>will not run</b> on other browsers. Download Chrome [HERE](https://www.google.com/chrome/?brand=JJTC&gclid=CjwKCAjwh-CVBhB8EiwAjFEPGWBaOaQ9LG5rsIbJgZUnCLCSpGf9A2d5NhNmskwjCH0lNp87Ifs0mBoCbsEQAvD_BwE&gclsrc=aw.ds)
2. Download this repository as a .zip file and extract it inside a folder.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/gitDownload.png' style='height: 400px'>
3. Open Google Chrome and click on the <i>Extensions Icon</i>. It is displayed as a small puzzle piece next to the search bar.<br><img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/tutorialExtensionsIcon.png' style='height: 400px;'>
4. This will bring up a popup window, find the <i>Manage Extensions</i> icon and click on it.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/tutorialExtensionsPopup.png' style='height: 400px;'>
5. This will open a new tab that will show you Chrome's Extensions Settings. Make sure you have the Developer Mode <i>enabled</i>.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/devMode.png' style='height: 400px;'>
6. On this tab click <i>Load unpacked</i>.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/loadUnpacked.png' style='height: 400px;'>
7. This will make you file browser popup. Select the folder where you extracted the extension and click <i>Select Folder</i>.
8. You should now see the YT Bookmarker extension appear.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/ext.png' style='height: 400px;'>

### Pin the extension for easy access
1. After you have loaded the extension click on the <i>Extensions</i> Icon again.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/tutorialExtensionsIcon.png' style='height: 400px;'>
2. Find the YTBookmarker extension and click on the <i>Pin</i> Icon.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/pinExt.png' style='height: 400px;'>
3. The Extension is now pinned!

### How to Use
1. Navigate to YouTube and open a video.
2. You should see the <i>Add Bookmark</i> Icon next to the left YouTube controls. Click on it to save the current timestamp.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/tutorialIcon.png' style='height: 400px'>
3. To view your saved bookmarks simply click on the extension icon next to Chrome's search bar.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/openUi.png' style='height: 400px'>
4. You can now view your Saved Bookmarks!
5. Click on the <i>Play</i> icon, to instantly navigate to the selected bookmark, or click the <i>Delete</i> icon to delete a bookmark.<img src='https://github.com/ArchontisKostis/yt-bookmarker-extension/blob/v1.0.0/assets/uiControls.png' style='height: 400px'>
